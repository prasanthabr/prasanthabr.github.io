# Notes on Designing Strategy

A set of choices made today in the face of uncertainity to be winning in the future. 

There is a process that was developed that works for companies of different scale which produces actionable steps that generate impact that is more than incrementally.

Strategy Process Map
--------------------
7 step process

_notes_
State (Identify) the problem
rearticulate this as a question
generate possibilities (the idea is to generate not decide)
loop thru possibilities and identify what needs to be true to make this a winning strategy
find the barriers or obstacles for these possibilities
test (for each possibility): - research, prototype, look at industry data for patterns <--
Select an option based on what we found above
