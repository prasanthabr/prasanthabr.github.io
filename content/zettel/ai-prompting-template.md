#### context
From the Google Prompting Essentials Course

Good Prompting Template  

- Task
    - Persona
    - Format
- Context
- Reference
- Evaluate
- Iterate

The input content matters; the order of the content does not usually matter.  
However, there is advantage in iteratively asking questions to give the ai
clearer context to work with. This aligns with the AI token concept. More tokens
used, means more thinking -- roughly.
